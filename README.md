
📌 Overview
This project is a Process Mining Dashboard developed using Python. 
It analyzes event log data and visualizes business processes to understand how activities flow in a system. 
The dashboard helps identify bottlenecks, delays, and performance issues in workflows.

🎯 Features
Event Log Analysis – Reads and processes event log datasets.
Process Visualization – Displays workflow patterns using charts.
Bottleneck Detection – Identifies activities that slow down the process.
Activity Time Analysis – Shows time taken for each activity.
Interactive Dashboard – Easy-to-understand visualization of process data.

🛠️ Technologies Used
Python – Main programming language.
Pandas – Data processing and analysis.
Matplotlib – Data visualization.
Streamlit – Building the interactive dashboard.
NumPy – Numerical operations.

🚀 How to Run
Clone the repository:

git clone https://github.com/eekshithajami/process-mining-dashboard.git

Navigate to the project folder:

cd process-mining-dashboard

Install required libraries:

pip install pandas matplotlib streamlit numpy

Run the dashboard:

streamlit run dashboard.py
